# KinoXP

KinoXP er et simpelt biografsystem udviklet i Spring Boot.

## Funktioner
- Se filmforestillinger
- Klik på en film og se forestillinger
- Reservere billetter
- Se reservationer

## Taknologi
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- HTML, CSS, JavaScript


## Strukturk
Backend: Spring Boot REST API  
Frontend: statiske HTML sider
Database: MySQL
